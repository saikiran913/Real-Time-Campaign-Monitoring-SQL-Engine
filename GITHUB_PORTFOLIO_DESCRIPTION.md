# GitHub Portfolio Description

## Short Repo Description

SQL-first campaign monitoring engine built in SQLite with KPI calculations, budget pacing, anomaly detection, alerts, health scores, and investigation scenarios.

## Longer About Section

This project simulates a real-time marketing campaign monitoring system using synthetic campaign data and SQLite. It demonstrates a full SQL analytics workflow: raw data loading, staging and cleaning, KPI calculations, budget pacing, trend monitoring, alert generation, campaign health scoring, validation, and investigation queries.

I built a local SQL analytics engine that simulates real-time campaign monitoring. The goal was to show SQL data engineering, data quality, reconciliation, and business-focused analytics design in one end-to-end project.
