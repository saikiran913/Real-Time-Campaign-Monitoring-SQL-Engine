# Investigations

This folder contains SQL investigation scenarios for common campaign monitoring issues.

## Scenarios

- `scenario_01_high_spend_zero_conversion.sql`: Finds campaigns spending heavily with zero conversions.
- `scenario_02_roas_drop.sql`: Reviews ROAS drops or ROAS below target.
- `scenario_03_budget_overspend.sql`: Reviews campaigns overspending against budget pacing.
- `scenario_04_ctr_drop.sql`: Reviews CTR below target or traffic drops.
- `scenario_05_campaign_no_impressions.sql`: Reviews active campaigns with no impressions.
- `scenario_06_critical_campaigns.sql`: Lists critical and at-risk campaigns.

## Real Troubleshooting Value

These scenarios represent practical data engineering and business review questions. They help move from raw alert outputs to focused analysis that a marketer or analyst could act on.

## How To Use In A Portfolio Walkthrough

Run the full SQL pipeline, open a scenario file, explain the business question, run the query, and describe the recommended action.
