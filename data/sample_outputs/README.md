# Sample Outputs

This folder contains sample output queries for GitHub visitors and portfolio walkthroughs.

## File

- `sample_output_queries.sql`

## How To Run

1. Run the full SQL pipeline through `sql/07_anomaly_detection_and_alerts.sql`.
2. Open `data/sample_outputs/sample_output_queries.sql` in DB Browser for SQLite.
3. Run one query at a time or run the full file.

- Budget pacing summary

- Executive-style final summary
